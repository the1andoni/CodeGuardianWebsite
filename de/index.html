CodeGuardian

Ihr unverzichtbares Tool für sichere und effiziente Codeverwaltung.

Über CodeGuardian

CodeGuardian ist ein Bot, der entwickelt wurde, um automatisierte Aufgaben auf GitHub zu erleichtern. Er bietet Funktionen wie Pull-Request-Überwachung, Code-Qualitätsprüfung und Sicherheitsprüfungen.

Funktionen

  • Automatische Überwachung von Pull Requests
  • Code-Qualitätsprüfung mit flake8
  • Sicherheitsprüfung mit bandit
  • Discord-Benachrichtigungen
  • Slash-Befehle für schnelle Aktionen
  • JSON-basierte Persistenz zur Vermeidung doppelter Benachrichtigungen
  • Detailliertes Logging aller Ereignisse
