About CodeGuardian

CodeGuardian is a bot designed to automate tasks on GitHub. It provides features like pull request monitoring, code quality checks, and security scans.

Features

  • Automatic monitoring of pull requests
  • Code quality checks with flake8
  • Security scans with bandit
  • Discord notifications
  • Slash commands for quick actions
  • JSON-based persistence to avoid duplicate notifications
  • Detailed logging of all events
